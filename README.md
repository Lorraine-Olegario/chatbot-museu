# Chatbot Museu

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 14.0.0.

Este projeto foi desenvolvido em paralelo ao live code, visando aprimorar e desenvolver habilidades em Angular. Trata-se de um chat bot destinado a fornecer informações e tirar dúvidas dos visitantes do Museu do Doce de Pelotas.

Tire Dúvidas: O chat bot está equipado para responder perguntas frequentes sobre o museu, seus horários de funcionamento, exposições e muito mais.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.
